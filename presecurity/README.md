# 📚 TryHackMe Pre-Security Certification Repository

This repository contains study materials and cheatsheets for the TryHackMe Pre-Security certification.

## 📁 Repository Structure

```
presecurity/
├── README.md                  # 📋 Main repository documentation
├── CHEATSHEET.md              # 📚 Comprehensive cheatsheet
├── modules/                   # 📁 Module-specific content
│   ├── 01-cybersecurity.md     # Introduction to Cybersecurity
│   ├── 02-networking.md       # Networking Fundamentals
│   ├── 03-web.md              # How the Web Works
│   ├── 04-computer-fundamentals.md # Computer Fundamentals
│   ├── 05-operating-system-basics.md # Operating System Basics
│   ├── 06-software-basics.md  # Software Basics
│   └── 07-attacks-and-defenses.md # Attacks and Defenses
└── references/                # 📖 Reference materials
    ├── ports-protocols.csv     # Common ports and protocols
    ├── ip-ranges.csv           # IP address ranges
    └── dns-records.csv         # DNS record types
└── .gitignore                 # Git ignore file
```

## 🚀 Getting Started

1. Clone this repository:
   ```bash
   git clone https://github.com/yourusername/presecurity.git
   ```

2. Navigate to the repository:
   ```bash
   cd presecurity
   ```

3. Open the cheatsheet:
   ```bash
   open CHEATSHEET.md
   ```
